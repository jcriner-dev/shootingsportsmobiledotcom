SSMI v0.3.0 Released

19 Sep 2014

We've released v0.3.0 of both the server program and the scoring application for IDPA ™. Below is a short list of changes:

  • Simplified install process.
  • New install help file.
  • New bundled executable files for Windows, Mac, and Linux.
  • New flexible licensing options.
  • New server messages on startup related to licensing and ip addresses.
  • New Terms and Conditions including open source software notices.
  • New SQL command interface to interact with the database directly.
  • New SQL file to generate a standard set of statistics from your database.
  • New check for valid IDPA numbers on signup and when shooting a classifier.

Get it while it's hot! We've been using this software for two (2) years at our local club. It works. It is reliable. We score matches as fast or faster than on paper with fewer errors. And, it has saved us countless hours scoring matches and distributing results.
