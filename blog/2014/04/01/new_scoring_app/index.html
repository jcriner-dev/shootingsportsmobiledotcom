SSMI Score v0.2.0 Released

01 Apr 2014

We've released v0.2.0 of the scoring application for IDPA ™. Most of the changes are under the hood. Below is a short list of changes the user will notice:

  • There are now installable apps for iOS and Android suitable for the App Store and Google Play.
  • A new theme which features:
    • Largely black and white colors for better outdoor visibility.
    • Bigger fonts.
    • Flat and more responsive buttons and dropdowns.
  • You now edit the squad order by dragging left to move up, and right to move down.
  • Times and downs inputs automatically translate any disallowed character into a comma allowing the use of number or telephone style keypads.
  • There are tighter controls on the score page which prevent you from leaving the page, changing stages, or changing shooters unless the current data on the page is already saved or blank.
  • The Upload and Delete Data pages now itemize the data to be uploaded or deleted so SO's can better review the information before taking action.
