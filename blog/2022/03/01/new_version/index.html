SSMI v2022.1.1 Released

01 Mar 2022

We've released v2022.1.1 of the server program for IDPA ™. See CHANGELOG.md for a full list of changes.

[2022.1.1] - 2022-03-01

Added

  • Added new matchsignup properties to db/csv/shooter_properties.csv.
  • Added new sql files for major matches to db/sql.

Changed

  • Changed server build system.
    • Using aws build system for windows.
      • "name": "Windows_Server-2012-R2-English-STIG-Full"
  • Changed scoring app build system.
    • Using local build system for android.
      • "cordova": "8.1.2"
      • "cordova-android": "^7.1.4"
      • "android-minSdkVersion": "19"
      • "android-targetSdkVersion": "27"
      • Supports android versions >= 4.4

Other

  • Scoring versions >= 2020.1.1 are fully compatible with this admin and server version.
