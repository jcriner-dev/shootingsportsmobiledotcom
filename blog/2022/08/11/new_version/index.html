SSMI v2022.2.3 Released

11 Aug 2022

We've released v2022.2.3 of the server program for IDPA ™. See CHANGELOG.md for a full list of changes.

[2022.2.3] - 2022-08-11

Added

  • Added custom sql files for the 2022 MI State Match.

Changed

  • Change all "Custom Properties" defaults to "Yes".
  • Changed update_match_dnf.
    • Match DNF is calculated based on either the specified stage count in the match description, or the most common stage count for all competitors when not specified.
    • Previously, it was recalculated when running match results reports or exports.
    • Now, it is recalculated on every upload. This allows for chrono or equipment DNF's to be manually applied after all uploading is done.
  • Changed font size calculation for pdf reports to better use available space with greater than 10 stages.

Fixed

  • Fixed most_accurate to require hnt = 0.
  • Fixed display of dnf and dq in match results and match exports.
  • Fixed bug in patch_db when comparing versions.

Other

  • Scoring versions >= 2022.2.2 are fully compatible with this admin and server version.
