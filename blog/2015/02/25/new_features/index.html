New Features

25 Feb 2015

The following updates were made to the IDPA ™ version of the software:

  • 2015 Rule Changes
    • Added REV & CCP divisions
    • Removed ESR & SSR divisions
    • Allow classification in BUG division
    • Changed import / export / IDPA functions to handle division changes
  • Added match performance promotion feature
  • Added custom shooter properties (Shirt, Lunch, etc)
  • Added match report by stage
  • Added capability to change "Shooter Agree?" message in scoring app
  • Added random alias to new shooters created in scoring app
  • Added option to prevent updating existing shooters when importing
  • Added license management via web interface
  • Fixed login cookie bugs
  • Added embedded device support
    • Device becomes a wifi hotspot on boot
    • Accessible over wifi
      • via fixed ip address
      • via name if clients support Zeroconf networking
        • iOS, Linux, Mac out of the box
        • Windows requires Apple Bonjour installed
        • Android 4.1 does not work
    • Accessible over ethernet
      • via dhcp ip address
      • via name if clients support Zeroconf networking
        • iOS, Linux, Mac out of the box
        • Windows requires Apple Bonjour installed
        • Android 4.1 does not work
    • IDPA functions work when ethernet is plugged in to internet router
    • Run ssmi_server on boot
    • Cron job keeps ssmi_server running if process dies unexpectedly
    • Added feature detection for embedded device and real time clock
    • Added automatic clock syncing to client on login
    • Added software reboot / shutdown feature
      • Required to safely reboot / shutdown embedded devices without a power off button
