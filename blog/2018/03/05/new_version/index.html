SSMI v0.5.5 Released

05 Mar 2018

We've released v0.5.5 of both the server program and the scoring application for IDPA ™. Below is a list of changes:

[0.5.5] - 2018-03

Added

  • Added PCC and CO as standard divisions.
  • Added Admin SQL Result page button to copy results to clipboard.

Changed

  • Changed classifier signup to have a method (STD, ABBR, PCC). See the IDPA rules regarding classifiers for more information.
  • Changed classifier results reports to have a method (STD, ABBR, PCC).
  • Changed classifier export to return a ZIP file containing CSV files where the classifiers are grouped by date and method.
  • Changed classifier export CSV files to use the current "classifier with scores" template instead of an old format which was still recognized.
  • Changed results reports footers based on type.
  • Changed Scoring App Support
    • Build process required phonegap-version bump from cli-5.4.1 to cli-6.5.0
      • Minimum Android version = 4.1
      • Minimum iOS version = 8.0
  • Changed Admin and Scoring UI from "Shooter ID" to "IDPA ID" for clarity and consistency.

Fixed

  • Fixed bug in CSS where colored backgrounds were covering background images (yellow and dropdown arrow).
  • Fixed bug which prevented highlight / copy on Admin SQL Result page.
