Updated Help, Added Categories, and Better Messages

01 Oct 2013

The following updates were made to the IDPA ™ version of the software:

  • Online help files were updated to be consistent with the latest version of the software.
  • The 2013 categories were implemented (along with High Safety Officer). Using categories is optional for match and season standings reports.
  • Better confirm messages were added to the upload and delete process.
