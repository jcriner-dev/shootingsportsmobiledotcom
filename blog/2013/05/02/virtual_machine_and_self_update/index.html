Software Deployment

02 May 2013

For the near term, we've opted to deploy the server software as a guest appliance that can be imported in to VirtualBox. VirtualBox runs on Windows, Linux, Macintosh, and Solaris hosts. So deployment consists of the following steps:

  1. Install VirtualBox.
  2. Get the latest Shooting Sports Mobile appliance file for your sport.
  3. Import the appliance file in to VirtualBox.
  4. Open the proper firewall ports in your host machine's firewall.
  5. Start the virtual machine.

At this point you can start the web browser (we recommend Firefox) on your host computer, or any other networked computer or device, and go to the Admin web page for Shooting Sports Mobile. There is also a Software Update page in the Admin interface that allows you to update your system to the latest version without downloading an entire virtual appliance.

When you are ready to score on your mobile devices, simply clear the Shooting Sports Mobile web cache on your mobile device and point your device's web browser to the scoring app web page address on your server computer.

In the long term, we are looking at several different deployment options to simplify installation of the server software and reduce the size of the files required.
