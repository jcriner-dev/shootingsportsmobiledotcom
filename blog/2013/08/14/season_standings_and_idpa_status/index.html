Season Standings and IDPA Status

14 Aug 2013

Code development has resumed on the IDPA ™ version of the software, and we finalized the season standings reports. We also implemented the code to download and import IDPA ™ member status (membership expiration date and current classifications). This allows us to factor those items in when determining season standings eligibility.

Also added were code changes that will automatically become consistent on 1 October 2013 when the new rule goes into effect. This includes changes to shooter categories and classification times.
