SSMI v0.5.2 Released

16 Mar 2017

We've released v0.5.2 of both the server program and the scoring application for IDPA ™. Below is a list of changes:

[0.5.2] - 2017-03-16

Fixed

  • Fixed networking bug in scoring app on Android.
  • Fixed server code converting booleans to integers (code -> database) and vice versa.
  • Fixed server code deleting / editing match stages.
