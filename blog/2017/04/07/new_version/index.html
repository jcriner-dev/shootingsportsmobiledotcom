SSMI v0.5.3 Released

07 Apr 2017

We've released v0.5.3 of both the server program and the scoring application for IDPA ™. Below is a list of changes:

[0.5.3] - 2017-04-07

Changed

  • Changed all versions to 0.5.3 for consistency.
  • Changed match performance promotions to include DNF and DQ per IDPA Match Admin v1d 2017-03-21 M-4.1.
  • Changed match performance promotions to be a multiple of 9 per IDPA Match Admin v1d 2017-03-21 M-4.1.
