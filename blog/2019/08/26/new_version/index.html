SSMI v2019.1.4 Released

26 Aug 2019

We've released v2019.1.4 of both the server program and the scoring application for IDPA ™. See CHANGELOG.md for a full list of changes.

[2019.1.4] - 2019-08-26

Added

  • Added sql files for mailman list checking.
  • Added sql files for safety officer list generation.

Fixed

  • Fix datetime_utils isoparse timezone offset parsing bug.

Other

  • Admin and scoring versions 2019.1.3 are fully compatible with this server version.
