SSMI v0.5.8 Released

19 Feb 2019

We've released v0.5.8 of both the server program and the scoring application for IDPA ™. Below is a list of changes:

[0.5.8] - 2019-02

Changed

  • Upgrades for CO division and new classifier times.
  • Upgrades for PCC division and new classifier methods.
  • Changed member status download / import process to work with new IDPA website.
  • Changed classifier export format for upload to new IDPA website.
