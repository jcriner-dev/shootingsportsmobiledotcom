SSMI v2019.1.3 Released

22 Apr 2019

We've released v2019.1.3 of both the server program and the scoring application for IDPA ™. Below is a list of changes:

[2019.1.3] - 2019-04-22

Added

  • Added --version flag.
  • Added better name parsing for IDPAHQ status files.
  • Added method to convert IDPAHQJSON2019 to IDPAHQCSV2019 (cli only).

Changed

  • Changed all html, js, and styl code formatting to prettier.

Fixed

  • Fixed bug in admin classifier stage edit when finding method in sql.
  • Fixed bug in admin user edit when deleting user.
  • Fixed db/divisions.csv to only include XXX since PCC is now official.

[2019.1.2] - 2019-04-17

Changed

  • Changed all python code formatting to black.
  • Changed db/sql/classifier_from_match.sql to export format consistent with admin classifier export function.

Fixed

  • Fixed IDPA classifier exports to have correct fields and values for upload to idpa.com.
  • Fixed handling of classification dates when importing IDPA member status files.

[2019.1.1] - 2019-04-05

Changed

  • Changed from "semver.org" to "calver.org" versioning.
    • Rationale
      • Project should have been past version 1.0 over five years ago.
      • Project did not follow strict semver regarding breaking changes.
      • Rules tend to be season oriented.
    • Format
      • YYYY.version.patch
      • YYYY = four (4) digit year.
      • version = incremental integer version for the year.
        • Used for feature changes or improvements.
        • MAY or MAY NOT be backward compatible with the same YYYY.
      • patch = incremental integer bugfix.
        • Used for bugfixes only.
        • MUST be backwards compatible with the same YYYY.version.

Fixed

  • Fixed filename bug in idpahq.py when using --diff-status on the command line.

[0.5.8] - 2019-03-04

Added

  • Added match export for upload to new IDPA website.
  • Added Database / Shooters / Export IDPA Number feature to get a list of IDPA numbers for member lookup at new IDPA website.
  • Added sql file to get a list of IDPA numbers for member lookup at new IDPA website.

Changed

  • Upgrades for CO division and new classifier times.
  • Upgrades for PCC division and new classifier methods.
  • Changed classifier export format for upload to new IDPA website.
  • Changed member status download process to work with new IDPA website.
  • Changed member status check / import process to better handle additional file formats.
  • Changed "Emergency Contacts" features.
    • Shooter export / import file format now handles emergency contacts.
    • Database / Shooters / Export Emergency Contacts
    • Allow modification via:
      • Import with custom python script.
      • Import Shooters (1 Emergency Phone).
      • Shooters (1 Emergency Phone).
      • Signup (1 Emergency Phone).
