About

Shooting Sports Mobile brings mobile electronic scoring to amateur and professional shooters alike. It was created by shooters for shooters and because we would rather practice shooting instead of data entry. The scoring system has been successfully used under real world conditions since the beginning of 2013. Details for acquiring the scoring system can be found at our online store. If you are interested in learning more, please send an email to support@shootingsportsmobile.com.

You can find screenshots, tutorials, and the FAQ further down the page.

Who:

James Criner, IDPA ™ shooter and engineer.

What:

  • A mobile scoring system for IDPA ™. Versions for other sports may be added pending interest.
  • Software that is developed according to the following guiding principles:
    • Simple user interfaces are better than complex ones.
    • Doing a few things well is better than doing a bunch of things poorly.
    • Scoring should represent a small fraction of the effort required to run a match.
  • The server program runs on Windows, Mac, and Linux operating systems and can be installed on a desktop / laptop computer, hosted on the internet, or run on an embedded mini-computer.
  • The admin web interface runs in an up-to-date desktop or mobile web browser.
  • The scoring program can be run as a native app on iOS and Android; or through the admin web interface.

Features:

  • CHANGELOG.md
  • Always up to date with the latest IDPA rules.
  • Scoring App
    • The mobile device can be disconnected from the network (airplane mode or wifi-off) and still record stage scores.
    • Add shooters to your squad from signup lists or as brand new shooters right on the device.
    • Edit the squad order and add / remove shooters from your squad at any point in the match.
    • The next shooter in the squad is automatically brought up once you confirm the current shooter's stage score.
    • You can change the first shooter up on each stage and still cycle through the same sequence.
    • Times (and Points Down) are numerically entered via the keypad.
    • Penalties are tap to increment and hold to reset.
    • Stage scores are summarized, the shooter must agree (verbally or by physically clicking a button), and they are confirmed by the scorer during the save process. This should suffice for major matches as it rquires a shooter to agree with their score and allows them to write down their own carbon copy if they choose.
    • Customizable score confirmation message.
    • Partial matches can be scored on multiple devices (when people change squads mid-match).
    • IDPA ™ classifier matches, if valid, have their result reported immediately on the mobile device.
    • Upload results to the main scoring computer.
  • Admin Web Interface:
    • Score on the fly and generate HTML or PDF reports for matches (standard or by stage), IDPA ™ classifier results, and season standings.
    • Create or Edit matches (including name, description, number of stages, stage descriptions, and round count).
    • Create or Edit shooters (name, address, email, phone, sport id number).
    • Create or Edit unofficial extra divisions.
    • Add custom shooter properties such as "Shirt" or "Lunch".
    • Create signup lists for faster squadding on your mobile device. New shooters are created automatically during the signup process.
    • Edit a shooter's categories (lady, military, etc) during signup or via the "Shooters" page.
    • Edit stage results after the fact in a spreadsheet style interface.
    • Create or Edit password protected user accounts that have any number of powers such as uploading results, getting match reports, and using the admin interface.
    • Import makes, models, and categories to the existing database.
    • Import shooter info from a CSV file. Yes, you can import your shooters from other scoring software.
    • Export shooter info to a CSV file (for spreadsheets or email lists).
    • Export classifier results to a CSV file suitable for upload to IDPA HQ ™.
    • Download and import shooter status and classifications from IDPA HQ ™.
    • Backup and Restore the main database.
    • Perform software updates.
    • Edit options like club details or limits on times, points down, and finger or muzzle warnings.
    • Bundled executable files for Windows, Mac, and Linux.
    • Flexible licensing options.
    • Messages on startup related to licensing and ip addresses.
    • SQL command interface to interact with the database directly.
    • SQL file to generate a standard set of statistics from your database.
    • Check for valid IDPA numbers on signup and and when shooting a classifier.

Real World Usage:

  • Signup and Match Admin on multiple computers simultaneously across the network.
  • Scoring on iPhone, iPad, and Android tablets.
  • 76 matches, 319 classifiers, and 50422 stages scored under live fire conditions.

Scoring Workflow:

  • Start the server program that acts as both a lightweight web server and stores data in a database.
  • Create a match and signup shooters via desktop and / or mobile web browsers.
  • Launch the scoring app and download signup lists and known shooters on your mobile devices.
  • Create your squads and score matches on your mobile devices.
  • Upload the stage results to the database.
  • Go to the results web page to score the match and get PDF match reports.

FAQ:

Q1: Can I get the scoring app from a mobile app store?

A1: Native apps are available for iOS and Android. Please check the online store for more information.

Q2: How do I get the scoring system and how much does it cost?

A2: Please check the online store for more information. If you have additional questions, please send an email inquiry to support@shootingsportsmobile.com.

Q3: What are the requirements for running the system?

A3: 1) A computer to run the server program, 2) A way to connect the server computer and the mobile devices to the same network (e.g. a wireless router or running a laptop with wifi in ad-hoc mode), 3) As many mobile devices or laptops as you have squads.

Q4: Do I need to be connected to wifi or the internet out on the range?

A4: No. You only need to connect your mobile device to the same network the server computer is on in order to download the most recent signup lists and match information. This is usually done just before the match. You can then safely disconnect the mobile device from the network (wifi off or airplane mode) to conserve battery life and still score out on the range.

Q5: Can I import my shooter database from my existing scoring software?

A5: Yes. Export your shooters from your existing scoring software to a CSV (Comma Separated Values) file. Import that file in to Shooting Sports Mobile. You can import CSV files from certain scoring software directly. Others may require a little massaging in a spreadsheet first. But, in general, if you can create a CSV file of shooter data that has the right columns then you are good to go.

Q6: Will the scoring app run on your mobile device?

A6: If you intend to use devices running iOS >= 8.0 or Android >= 4.1, the answer is most likely YES. We are running iPhone 6+ and inexpensive 7" Android tablets with great results at our local club.

Q7: Are E-Ink or E-Paper displays supported on mobile devices?

A7: No. We do not support these type of black and white displays as they may experience an excessive amount of "flickering" during use. It is a technical trade-off between display performance, visibility, and user experience. Although the scoring app was not designed for, and is not supported on, these type of displays, feel free to give it a try. Everyone's visual needs and tolerances are different.

Q8: Will the admin web interface run on your web browser?

A8: See the table below. If you have any issues whatsoever, please install and use a current desktop version of Firefox or Chrome.
